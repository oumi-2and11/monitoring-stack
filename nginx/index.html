Prometheus + Grafana 监控集群

Docker Compose 一键部署 · Nginx 反向代理 · 网络隔离安全加固
Grafana 可视化面板
监控数据可视化大屏。包含「主机与容器概览」和「Flask 服务指标」两个 Dashboard，自动加载无需手动配置。
/grafana/
Prometheus 指标采集
核心采集引擎。查看所有采集目标（Targets）状态、告警规则（Rules）及触发情况、静态与动态服务发现配置。
/prometheus/
Alertmanager 告警管理
告警路由与分组中心。负责接收 Prometheus 触发的告警，按规则分组后推送到 Webhook 接收器。
/alertmanager/
告警接收日志
自研 Webhook 接收器。以表格形式展示已收到的所有告警记录，包含告警名、级别、实例、触发时间等。
/alert-receiver/
Flask 被测服务
模拟业务服务，提供正常 /、错误 /error、慢请求 /slow 三个接口，以及 Prometheus 指标端点 /metrics。
