模拟服务器错误

此接口故意返回 HTTP 500 状态码，用于模拟服务器内部错误场景。

在 Grafana 的 Flask Service Overview 面板中，可以看到 5xx 错误率上升；当错误率超过 5% 并持续 1 分钟时，将触发 FlaskAppErrorRateHigh 告警。

触发方式

手动访问：连续多次访问本页面

PowerShell 脚本：.\scripts\trigger_error.ps1

脚本会循环调用 /error 接口约 2 分钟，同时混入 20% 的正常请求，使错误率约为 80%，从而稳定触发告警。

HTTP Response
Status: 500 Internal Server Error { "status": "error", "message": "Internal server error" }
